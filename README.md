# Installation steps

- Cloning Project 
  ```console
    git clone https://github.com/thanhnd0101/DepotApp_BE.git
  ```
- Gem install
  ```console
    bundle install
  ```
- Start server
  ```console
    rails server
  ```
# Unit test
